# Инструкция по безопасному обновлению macOS

## Ваша текущая система
- **Версия macOS:** 12.7.6 (Monterey)
- **Model Identifier:** MacBook9,1
- **Требуется:** macOS 14.0.0 или выше

## ⚠️ ВАЖНО: Перед обновлением

1. **Создайте резервную копию**
   - Используйте Time Machine или другой метод резервного копирования
   - Убедитесь, что резервная копия завершена успешно

2. **Проверьте совместимость**
   - MacBook9,1 (MacBook 12" 2016) поддерживает macOS до версии 12.x (Monterey)
   - Для обновления до macOS 14+ может потребоваться более новая модель Mac
   - Проверьте официальную совместимость на сайте Apple

3. **Освободите место на диске**
   - Требуется минимум 20GB свободного места
   - Рекомендуется 30-40GB для комфортной установки

## Способы обновления

### Способ 1: Через App Store (Рекомендуется)

1. Откройте **App Store**
2. Найдите последнюю версию macOS (например, macOS Sonoma или macOS Sequoia)
3. Нажмите **"Загрузить"** или **"Получить"**
4. Дождитесь завершения загрузки (может занять 30-60 минут)
5. Установщик откроется автоматически
6. Следуйте инструкциям на экране

**Или используйте скрипт:**
```bash
./update_macos.sh
```
Выберите опцию 1 для открытия App Store.

### Способ 2: Через Системные настройки

1. Откройте **Системные настройки** (или **Настройки системы** в новых версиях)
2. Перейдите в **Обновление ПО** (или **Обновление программного обеспечения**)
3. Если доступны обновления, нажмите **"Обновить сейчас"**
4. Следуйте инструкциям

### Способ 3: Через терминал (для опытных пользователей)

```bash
# Проверка доступных обновлений
softwareupdate --list

# Установка всех доступных обновлений
sudo softwareupdate --install --all --agree-to-license

# Перезагрузка после установки (если требуется)
sudo reboot
```

## Проверка совместимости MacBook9,1

**MacBook 12" 2016 (MacBook9,1)** поддерживает:
- macOS Monterey (12.x) - ✅ Текущая версия
- macOS Ventura (13.x) - ⚠️ Возможно, но не официально
- macOS Sonoma (14.x) - ❌ Не поддерживается
- macOS Sequoia (15.x) - ❌ Не поддерживается

**Важно:** Если ваша модель не поддерживает macOS 14+, вы не сможете установить эту версию. В этом случае:
1. Оставайтесь на macOS 12.7.6 (последняя поддерживаемая версия)
2. Или рассмотрите обновление до macOS 13.x (если доступно)

## Альтернативное решение

Если приложение требует macOS 14.0.0+, но ваш Mac не поддерживает эту версию:

1. **Проверьте, есть ли версия приложения для macOS 12.x**
2. **Свяжитесь с разработчиками приложения** для запроса поддержки macOS 12.x
3. **Рассмотрите обновление Mac** (если это возможно)

## Безопасность

- ✅ Всегда создавайте резервную копию перед обновлением
- ✅ Подключайте Mac к зарядке во время обновления
- ✅ Не выключайте Mac во время установки
- ✅ Убедитесь в стабильном подключении к интернету
- ✅ Закройте все приложения перед началом обновления

## Что делать после обновления

1. Проверьте версию macOS: `sw_vers`
2. Установите обновления безопасности: `softwareupdate --install --all`
3. Проверьте работу всех приложений
4. Восстановите данные из резервной копии (если необходимо)

## Проблемы и решения

### Ошибка "OS X 14.0.0 is not supported"
- Это означает, что ваша модель Mac не поддерживает macOS 14+
- Оставайтесь на macOS 12.7.6 или обновите Mac

### Недостаточно места на диске
- Освободите место, удалив ненужные файлы
- Используйте утилиту "Хранилище" в Системных настройках

### Обновление не запускается
- Перезагрузите Mac
- Проверьте подключение к интернету
- Попробуйте обновить через App Store вручную
